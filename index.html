١. أساسيات الكمبيوتر وأنظمة التشغيل

💻 نظام Windows

أكثر نظام منتشر. لازم نفهم كيف يعمل من الداخل: الـ Registry (قاعدة بيانات الإعدادات)، إدارة المستخدمين (Users & Groups)، والـ Active Directory في بيئة الشركات.

🐧 أساسيات Linux

نظام مفتوح المصدر وهو البيئة الأساسية لأي سيبراني. لازم نفهم بنيته (File System) وإن كل شيء فيه عبارة عن ملف (File).

📁 إدارة الملفات والأوامر

كيفية التنقل بين المجلدات (cd)، عرض المحتويات (ls)، إنشاء وحذف الملفات (mkdir, rm)، وقراءة محتوى الملفات (cat, less).

🔑 Authentication (المصادقة)

إزاي النظام يتأكد إنك إنت الصح؟ (شيء تعرفه زي كلمة السر، شيء تملكه زي الهاتف، شيء فيك زي البصمة).

٢. شبكات الحاسوب (Networking)

🌐 IP Address

هو "رقم الهوية" لجهازك على الإنترنت (مثل 192.168.1.1). بدونك ما تقدرش تتصل بأي حاجة. فيه IP عام (Public) وخاص (Private).

📖 DNS (Domain Name System)

هو "دليل التليفونات" بتاع الإنترنت. بيحول أسماء المواقع السهلة (google.com) لأرقام IP يفهمها الكمبيوتر.

🔒 HTTP / HTTPS

البروتوكول اللي بينقل بيانات المواقع. HTTP النسخة العادية، و HTTPS النسخة المشفرة (S تعني Secure) اللي بتحمي كلمات السر والبيانات.

🔀 الراوتر والسويتشات (Routers & Switches)

الراوتر بيوصل بين شبكات مختلفة (بيوصلك بالإنترنت)، والسويتش بيوصل الأجهزة مع بعضها جوه شبكة واحدة (زي شبكة المنزل أو الشركة).

٣. تعلم Linux بعمق

⌨️ أوامر Terminal

التيرمنال هي قوتك. تتعلم إزاي تبحث في الملفات (grep)، تتحكم في النظام، وتوقف العمليات (kill).

🛡️ الصلاحيات (Permissions)

لينكس بيدي صلاحيات لكل ملف: قراءة (r)، كتابة (w)، تنفيذ (x). وتتغير بأمر chmod عشان تحمي الملفات من التعديل.

👥 إدارة المستخدمين والخدمات

إزاي تضيف يوزر جديد (adduser)، و إزاي تشغل وتوقف خدمات السيرفر (systemctl) زي خدمة الويب Apache.

🖥️ إدارة السيرفرات

إزاي تدير سيرفر لينكس عن بعد من خلال بروتوكول SSH، وتشوف استهلاك الرام والمعالج (top, htop).

٤. البرمجة والـ Scripting

🐍 Python

أهم لغة للسيبراني. بتستخدم لكتابة أدوات اختراق مخصصة، تحليل البيانات، وأتمتة المهام بسهولة.

🌐 JavaScript

لغة الويب. لازم تتعلمها عشان تفهم ثغرات المتصفحات زي XSS، وتعرف إزاي الموقع بيشتغل من قدام (Front-end).

📜 أساسيات Bash

لغة سكربتات لينكس. بتخليك تجمع أوامر كتير في ملف واحد عشان تعمل مهمة معقدة بضغطة زر واحدة.

٥. المفاهيم الأمنية وأنواع الهجمات

🎭 أنواع الهجمات

فهم هجمات الحرمان من الخدمة (DDoS)، هجمات الوسيط (MITM)، وبرمجيات الفدية (Ransomware).

🔒 التشفير (Cryptography)

تحويل البيانات لكود مش مفهوم. فيه متماثل (مفتاح واحد زي AES) وغير متماثل (مفتاحين Public & Private زي RSA).

🔑 كلمات المرور (Passwords)

إزاي الهاش (Hash) بيحمي كلمات السر، وإزاي الهاكرز بيكسروها بهجمات القاموس (Dictionary) أو القوة الغاشمة (Brute Force).

🧠 الهندسة الاجتماعية (Social Engineering)

اختراق العقل البشري موش الكمبيوتر. الخداع النفسي عشان توقع الضحية في فخ تسليم بياناته (Phishing).

📊 إدارة المخاطر (Risk Management)

تقييم التهديدات (Threats) والثغرات (Vulnerabilities) وتحديد الأثر لو حصل اختراق، عشان نضع خطة دفاع.

٦. اختبار الاختراق الأخلاقي (Pentesting)

🔍 جمع المعلومات (Reconnaissance)

الخطوة الأولى. جمع أكبر قدر من المعلومات عن الهدف (إيميلات، أسماء نطاقات، تقنيات مستخدمة) بدون ما الهدف يحس.

🛠️ فحص الثغرات (Vulnerability Scanning)

استخدام أدوات زي Nessus أو Nmap لاكتشاف المنافذ المفتوحة والخدمات اللي فيها ضعف.

🧱 تأمين الأنظمة (Hardening)

إزاي تقفل المنافذ الزائدة، تفعيل الجدران النارية (Firewalls)، وتحديث الأنظمة عشان نمنع الاختراق.

📝 كتابة التقارير (Reporting)

أهم مرحلة للعميل. كتابة تقرير يشرح الثغرات اللي اكتشفتها، خطورة كل واحدة، وإزاي تتصلح.

٧. أمن تطبيقات الويب (Web Security)

💉 SQL Injection

حقن أكواد SQL في مدخلات الموقع عشان تخدع قاعدة البيانات وتسرق بيانات المستخدمين أو تتجاوز تسجيل الدخول.

⚡ XSS (Cross-Site Scripting)

حقن سكربتات JavaScript خبيثة في صفحات الويب، تنفذ في متصفح الضحية عشان تسرق الكوكيز (Cookies).

🔄 CSRF (Cross-Site Request Forgery)

خداع المستخدم وهو مسجل دخوله في موقع، عشان ينفذ طلب (Request) غير مرغوب فيه ل موقع تاني.

🛡️ Authentication & Authorization

المصادقة (إنت مين؟) والصلاحيات (مسموحلك تعمل إيه؟). أخطاء هنا بتخلي الهاكر يدخل حسابات غيره أو يطلع أدمن.

🔌 API Security

حماية الواجهات البرمجية اللي بتوصل بين التطبيقات. التأكد من تشفير البيانات المتبادلة ومنع تسريب بيانات المستخدمين.

٨. الفريق الأزرق / الدفاع (Blue Team / SOC)

📝 تحليل السجلات (Log Analysis)

قراءة أرشيفات الويندوز (Event Viewer) ولينكس (var/log/) لمعرفة إذا كان فيه نشاط غريب أو محاولة اختراق.

🚨 أنظمة كشف التسلل (IDS/IPS)

أنظمة مراقبة الشبكة (مثل Snort و Suricata) اللي بتكشف الهجمات وتمنعها في نفس الوقت.

🚑 الاستجابة للحوادث (Incident Response)

إزاي تتصرف لما يحصل اختراق؟ الخطوات العلمية: الاحتواء (Containment)، الإزالة (Eradication)، والتعافي (Recovery).

📡 SIEM (إدارة معلومات الأمن)

تجيع كل سجلات الشركة في مكان واحد عشان تربط الأحداث ببعضها وتكتشف الهجمات المعقدة (باستخدام أدوات زي Splunk).

٩. التشفير وأمن المعلومات (Cryptography)

🔐 التشفير المتماثل (Symmetric Encryption)

استخدام نفس المفتاح للتشفير وفك التشفير (مثل AES). سريع جداً بس المشكلة في إزاي تبعت المفتاح بشكل آمن.

🔑 التشفير غير المتماثل (Asymmetric Encryption)

استخدام مفتاحين: مفتاح عام (Public) للجميع، ومفتاح خاص (Private) سري. (مثل RSA) آمن لتوصيل المفاتيح.

#️⃣ الـ Hashing (التجزئة)

تحويل أي بيانات لبصمة رقمية ثابتة (مثل MD5, SHA-256). بيتستخدم للتأكد إن الملفات اتغيرتش أو لتخزين كلمات المرور بأمان.

✍️ التوقيع الرقمي (Digital Signatures)

إثبات إن الملف أو الرسالة جاية من الشخص الصح ومحدش عبث بيها أثناء الإرسال، باستخدام التشفير غير المتماثل.

١٠. المختبرات والتدريب العملي (Hands-on Labs)

🚀 منصات التدريب (TryHackMe / HackTheBox)

منصات أونلاين تقدر تتدرب عليها على أجهزة وهمية بشكل قانوني وآمن. TryHackMe ممتاز للمبتدئين، و HackTheBox للمستوى المتقدم.

🏠 بناء Home Lab

إزاي تبني مختبر في بيتك باستخدام برامج زي VirtualBox أو VMware. عشان تعمل تجارب واختراقات من غير ما تضر جهازك الأساسي.

🕷️ أداة Burp Suite

أهم أداة لاختبار اختراق الويب. بتخليك توقف الطلبات (Requests) بين المتصفح والسيرفر وتعدل عليها عشان تختبر الثغرات.

🛠️ أداة Metasploit

أقوى إطار عمل (Framework) للاختراق. فيه آلاف الثغرات الجاهزة اللي تقدر تستخدمها لتجربة الاختراق على مختبرك.
